## BoxAPI学習用SampleApplication
※CLIでも良かったんですけど、画面で動いたほうがPG以外も分かりやすいのでそうしました。
<動かす手順>
1.フリーメール作成+Boxをフリーアカウントで作成
2.BOX DEVで検索してログイン、アプリ登録
3.Git CloneしてEclipse+Spring Boot環境でlocalhost:8080で動確
