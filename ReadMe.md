## BoxAPI学習用SampleApplication
※CLIでも良かったんですけど、画面で動いたほうがPG以外も分かりやすいのでそうしました。<br>
<動かす手順><br>
1.フリーメール作成+Boxをフリーアカウントで作成<br>
2.BOX DEVで検索してログイン、アプリ登録<br>
3.Git CloneしてEclipse+Spring Boot環境でlocalhost:8080で動確<br>
